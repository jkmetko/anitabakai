Welcome to GalleryJack: The Reboot


Thank you for downloading GalleryJack: The Reboot, by CodingJack. This help document will assist you with setting up your gallery.

Need this item customized? Contact CodingJack


Special Notes

  • Ajax Mode will only work online.
  • Each category can display a set of thumbnails. Each set of thumbnails much be the same size.
  • The HTML5 doc-type is not necessary for the banner to work on your web-page.
  • "Slide-both" and "slide-over" transitions are not supported in IE7 and will default to "fade".
  • Thumbnail circle animation is built with HTML5 Canvas and will not animate in browsers that don't support HTML5 Canvas.
  • The fonts used are Google Web Fonts and do not need to be downloaded.
  • Sometimes the social buttons won't load locally, but this won't happen once the gallery is online.
  • The html files load the minified script versions (except for the main settings file). Keep this in mind if you plan to change any of the external scripts.
